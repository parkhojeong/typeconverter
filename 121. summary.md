* Caution
- MessageConverter(HttpMessageConverter) doesn't use ConversionService
- MessageConverter convert instance <-> json
- internal use Jackson
- response body json formatting by Jackson formatter

ConversionService use @RequestParam, @ModelAttribute, @PathVariable, view template

* Reference
spring provide various converter for usage
- Converter -> basic type convert
- ConverterFactory -> all class hierarchy structure 
- GenericConverter -> detail implementation, can use target field annotation information
- ConditionalGenericConverter -> if condition true, use converter

spring provide overall converter
- String
- Number
- Boolean
- Enum
- ...

ConversionService
- collect all converters
- and choose suitable converter
- provide easy use
